# Security Policy

## Supported Versions

The following versions of this Python project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 3.12.x  | :white_check_mark: |
| 3.11.x  | :white_check_mark: |
| 3.10.x  | :white_check_mark: |
| < 3.10  | :x:                |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security issue, please open a issue in this repository.

### What to Expect

- **Initial Response**: You will receive an acknowledgment within 48 hours
- **Updates**: We will provide updates on the status every 5-7 days
- **Resolution Timeline**: We aim to release a fix within 30 days for critical vulnerabilities
- **Disclosure**: Once fixed, we will credit you in the security advisory (unless you prefer to remain anonymous)

### Vulnerability Assessment

- **Accepted**: If the vulnerability is confirmed, we will work on a fix and coordinate a disclosure timeline with you
- **Declined**: If the report is not considered a security issue, we will explain our reasoning and may suggest alternative reporting channels if appropriate

Thank you for helping keep this project secure!
